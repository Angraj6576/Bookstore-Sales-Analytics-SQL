# 📚 Bookstore Sales & Customer Analytics System (PostgreSQL)

## 📌 Project Overview
This project is a relational database system built using PostgreSQL to manage an online bookstore.  
It performs sales analysis, revenue tracking, customer insights, and inventory management using SQL queries.

---

## 🛠️ Technologies Used
- PostgreSQL
- SQL
- CSV Data Import

---

## 🗄️ Database Schema
The project contains three main tables:
- Books
- Customers
- Orders

---

## 🔍 SQL Concepts Used
- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- Aggregate Functions (SUM, AVG, COUNT)
- ORDER BY
- LIMIT
- COALESCE

---

## 📊 Business Insights Extracted
- Total revenue generated
- Top spending customer
- Most frequently ordered book
- Genre-wise total books sold
- Remaining stock after fulfilling orders
